# ⏳ TimeVault – Digital Time Capsule

## Overview

TimeVault is a modern web application that allows users to create digital time capsules containing messages, notes, and images that remain locked until a selected future date.

Users can preserve memories, personal goals, letters to their future selves, or important moments and unlock them automatically when the chosen date arrives. The application provides a clean and interactive experience with countdown timers, capsule management, and secure storage using browser local storage.

---

## Features

* Create digital time capsules
* Add titles and descriptions
* Upload and store images
* Select future unlock dates
* Real-time countdown timer
* Automatic capsule unlocking
* View locked and unlocked capsules
* Delete unwanted capsules
* Private and public capsule options
* Search and filter capsules
* Responsive design for all devices
* Data persistence using Local Storage

---

## How It Works

1. Create a new capsule by entering a title, message, and unlock date.
2. Optionally upload an image and choose privacy settings.
3. Save the capsule.
4. The capsule remains locked until the selected date.
5. A countdown timer displays the remaining time.
6. Once the unlock date arrives, the capsule automatically becomes accessible.
